******Convert_image_to_PDF******
This python program can combine multiple image files into a PDF file.


**Request to users**************
1:This python program requires two modules:

	^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
	import img2pdf
	import PyPDF2	
	^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
   Please install modules.

2:Open 'Convert_image_to_PDF_Ver*.py' and edit folder location.

	--Convert_image_to_PDF_Ver*.py--

	#Please Set location
	file_location = r'C:......................\Convert_to_pdf' 

	---------------------------------

**********************************
